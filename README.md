透過グリッド君 (Transparent Grid-kun)

Web会議の画面共有やデザイン確認をスムーズにする、座標表示付きの透過グリッドChrome拡張機能です。
<img width="1024" height="420" alt="image" src="https://github.com/user-attachments/assets/5f48d4a8-1119-4deb-822d-174682ea0109" />


🌟 特徴

座標付きグリッド: 横軸(A, B, C...)と縦軸(1, 2, 3...)のラベル付きグリッドを最前面に表示。

クリック透過: グリッドを表示したまま、背後のWebページを通常通り操作可能。

カスタマイズ: セルサイズ、色、透明度を自由に変更可能。

簡単操作: ツールバーのアイコンをクリックするだけで瞬時にON/OFF。

🚀 開発背景

Web会議で画面共有中、操作者に特定の場所を伝えるのは大変です。「もうちょっと右、いや左上…」という曖昧な指示を、「Eの4をクリックしてください」という明確な指示に変えるために開発しました。

🛠 インストール方法

このリポジトリをダウンロード（Download ZIP）して解凍します。

Google Chromeを開き、アドレスバーに chrome://extensions/ を入力します。

右上の「デベロッパー モード」をオンにします。

「パッケージ化されていない拡張機能を読み込む」ボタンをクリックし、解凍したフォルダを選択します。

📖 使い方

ブラウザのツールバーにある「透過グリッド君」のアイコンをクリックしてグリッドを表示します。

座標を使って、画面上の位置を相手に伝えます。

詳細な設定（色やサイズ）を変えたい場合は、アイコンを右クリック ＞「オプション」を選択してください。
